¿Quiénes Somos?

Desarrolladores de la página web

Este proyecto web fue desarrollado de manera colaborativa por un equipo de seis integrantes, combinando distintas habilidades en diseño, programación y gestión. El objetivo principal fue crear una solución funcional y eficiente que responda a las necesidades planteadas, reflejando el compromiso y la coordinación del grupo.

Herramientas de Uso Front-end

Cursor & Visual Studio Code

Se eligieron Cursor y VS Code como editores de código por su simplicidad y facilidad de uso, además de las extensiones que cada uno ofrece para mejorar la productividad. Esto permitió al equipo trabajar en un entorno flexible y rápido, adaptándose a diferentes necesidades de desarrollo sin complicaciones técnicas.

Github & Vercel

La combinación de GitHub y Vercel resultó muy cómoda porque al subir los archivos al repositorio, estos se desplegaban automáticamente en Vercel. Así, el proyecto quedaba disponible en línea de inmediato y podía visualizarse incluso desde dispositivos móviles, optimizando el flujo de trabajo y la accesibilidad.
