## 📌 Descripción del Proyecto

**Palmas Street** es una aplicación web full-stack de comercio electrónico que integra autenticación de usuarios, gestión de productos dinámicos, carrito de compras interactivo y una interfaz responsive. El proyecto combina:

- **Frontend moderno** con HTML, CSS y JavaScript vanilla
- **Backend robusto** con Node.js y API RESTful
- **Base de datos en la nube** con Supabase (PostgreSQL)
- **Despliegue automático** en Vercel con CI/CD
- **Control de versiones** profesional con Git & GitHub

---

## 🛠️ Stack Tecnológico

### Frontend
- **HTML5** - Estructura semántica y accesible
- **CSS3** - Diseño responsive con Flexbox/Grid
- **JavaScript (ES6+)** - Lógica interactiva del cliente
- **Dark Mode** - Soporte nativo para tema oscuro

### Backend
- **Node.js** - Runtime JavaScript del servidor
- **API REST** - Endpoints para manejo de datos

### Base de Datos
- **Supabase (PostgreSQL)** - Base de datos relacional en la nube
- **Autenticación** - Sistema seguro de login/registro

### Despliegue
- **Vercel** - Hosting y CI/CD automático
- **Git & GitHub** - Control de versiones y colaboración

## 🚀 Instalación y Uso

### Requisitos Previos
```bash
- Node.js (v14+)
- Git
- Cuenta en Supabase
- Cuenta en Vercel (opcional)
```

### Clonar el Repositorio
```bash
git clone https://github.com/Styngixx/Palmas-Street.git
cd Palmas-Street
```

### Instalar Dependencias
```bash
npm install
```

### Configurar Variables de Entorno
```bash
# Crear archivo .env
SUPABASE_URL=tu_url_supabase
SUPABASE_ANON_KEY=tu_clave_anonima
```

### Ejecutar en Desarrollo
```bash
npm start
# O si tienes servidor local
npm run dev
```

### Desplegar en Vercel
```bash
# Conectar repo GitHub a Vercel

```
